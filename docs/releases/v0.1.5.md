## What's new

- Added support for the updated Codex/ChatGPT Desktop app on Windows.
- Fixed launch failures caused by Windows blocking direct `ChatGPT.exe` starts from `WindowsApps`.
- Added packaged app activation fallback using the app identity when direct launch is denied.
- Avoided closing the internal `resources\codex.exe` helper process by mistake.
- Waits longer for the app renderer before injecting the RTL fix.
- Updated README and security notes to describe Codex/ChatGPT Desktop behavior.

## Verification

- `npm audit --audit-level=moderate`
- Release ZIP excludes `.git`, `.tools`, `node_modules`, `.cache`, and `tmp`.
