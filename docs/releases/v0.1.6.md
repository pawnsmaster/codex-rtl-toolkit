## What's new

- Added RTL rendering for Arabic-script Markdown files opened in the Codex side panel.
- Keeps English-only and code lines LTR inside the Markdown editor.
- Handles CodeMirror line recycling and large update batches without dropping RTL processing.
- Added per-line direction handling for fenced `text` and `markdown` blocks in chat.
- Handles plain Markdown blocks even when they contain no headings, lists, or syntax-highlight tokens.
- Rescans dynamically rendered chat blocks as soon as their lines are added.
- Removed the fixed five-second launcher delay and now injects as soon as the renderer is ready.
- Clarified support for Arabic-script RTL languages, including Arabic, Persian, Urdu, Kurdish, Pashto, and Sindhi.
- Updated the desktop package and browser extension versions to `0.1.6`.
